> Realized with   : Laravel 7 + VueJS

> Payment gateway : Stripe

> Video's host    : Vimeo

## Install

```sh
composer install
```
or

```sh
php composer.phar install
```
then :

```sh
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
```
